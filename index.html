. (dot) - matches any single character

* - matches zero or more of the preceding character

+ - matches one or more of the preceding character

? - matches zero or one of the preceding character

^ - matches the start of a string

$ - matches the end of a string

{n} - matches exactly n of the preceding character

{n,m} - matches between n and m of the preceding character

[] - matches any character inside the brackets

[^] - matches any character not inside the brackets

| - matches either the expression before or after the pipe

\ - escapes special characters, allowing them to be matched literally
